Matlab2cpp
----------
This package contains two parts:
* Python 2.7 runtime for AntLR
* Matlab2cpp Python library built upon AntLR
  to auto-translate Matlab to C++

Visit the AntLR web sites for more information:
http://www.antlr.org
http://theantlrguy.atlassian.net/wiki/display/ANTLR4/Python+Target

For more information on Matlab2cpp:
http://github.com/jonathf/matlab2cpp

Installation
------------
Requirements:
Python 2.7.3

Linux:
As root, run the following command:
$ python setup.py install

The executable 'mconvert' is now available from path.

Windows:
> python.exe setup.py install

The executable mconvert.py must be copied to project or added to
enviromental variables manuallu.
